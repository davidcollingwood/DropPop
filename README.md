DropPop
==========================

Pre-Install

1. Install npm (http://nodejs.org)
2. $ npm install -g cordova
3. $ npm install -g bower
4. $ npm install -g brunch

Installation

1. $ npm install
2. $ bower install
3. $ brunch watch

Build

1. $ cordova plugin add https://github.com/Wikitude/wikitude-phonegap
2. $ cordova platform add ios
3. $ cordova build
